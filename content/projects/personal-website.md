# Personal Website

This is the website you're currently viewing! A personal hub that brings together different aspects of my life - professional, creative, and personal.

## Motivation

I wanted a single place where I could:

- Share my professional CV and experience
- Showcase my photography from travels
- Write about projects and experiments
- Track my goals and progress

## Tech Stack

- **Framework**: Next.js 14 with App Router
- **Styling**: Tailwind CSS 4
- **Language**: TypeScript
- **Hosting**: Vercel
- **Domain**: leventkurtis.com

## Features

### Completed

- [x] Dark theme CV page with scroll-based background darkening
- [x] Photography gallery with album-based structure
- [x] Light theme for photography pages
- [x] Landing page with section cards
- [x] Markdown-based blog/projects system

### Planned

- [ ] Goals tracking page
- [ ] Light/dark theme toggle
- [ ] Scroll animations with Framer Motion
- [ ] SEO improvements

## Lessons Learned

Building this site has been a great exercise in:

1. **Next.js App Router** - Understanding layouts, dynamic routes, and server components
2. **Tailwind CSS 4** - Learning the new @theme directive and PostCSS setup
3. **Content Management** - Creating a simple markdown-based CMS

## Next Steps

Continue adding features and content as the site evolves into a true personal life hub.
